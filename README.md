# robin - A lightweight CSS framework

A light CSS framework. It's lacking some features, but it's perfectly capable of delivering simple websites - check out the [example webpage](https://robin-css.herokuapp.com/example)

## Documentation

All documentation available [here](https://robin-css.herokuapp.com)

## Installation

### From Source

You can compile the SASS using your compiler of choice (I recommend node-sass)

### Compiled Files

You can find the latest minified files [here](http://res.cloudinary.com/dxfgtjxiz/raw/upload/v1526041925/robin.min.zip)
